# 014-password-strength（パスワード強度チェッカー）

入力したパスワードの強度を、長さ・使用文字種・エントロピー(bit)・推定解読時間（目安）から判定し、
弱点の指摘と改善提案を表示する完全クライアントサイドのツールです。

## 特徴

- 入力したパスワードは**保存も送信もしません**（プライバシー核心）。診断はすべて端末内のJavaScriptで実行。
- 強度は「未入力 / 非常に弱い / 弱い / 普通 / 強い / 非常に強い」の6段階（バー＋ラベル表示）。
- よくある弱いパスワード（漏えいパスワードリスト頻出パターン）、連続文字（abc/123等）、
  3回以上の繰り返し文字を検出して指摘。
- 使用文字種（英大文字・英小文字・数字・記号）のチェックリスト表示。
- エントロピー(bit)と、1秒あたり10億回試行のオフライン総当たり攻撃を想定した推定解読時間（目安）を表示。
- ダーク/ライトテーマ切替、PWA対応（オフライン動作）。

## セルフ確認

- 強度判定: 空文字→「未入力」、"password"→「非常に弱い」（漏えいパスワード一致）、
  "Tr0ub4dor&3Xyz!"のような長く文字種混在のものは「強い」〜「非常に強い」になることを確認済み。
- 入力パスワードはlocalStorage・sessionStorage・送信のいずれにも保存されない（保存対象はテーマ/Proフラグのみ）。
